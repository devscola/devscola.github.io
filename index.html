Aprende a programar

Devscola es una escuela de programación situada en Valencia,
Avd Catalunya nº11, Entresuelo (GEEKSHUBS)

Solo aceptamos los mejores candidatos que son capaces de aprovechar mejor la oportunidad de aprender a programar con la ayuda de nuestros selectos mentores y teaching assistants. Al finalizar el periodo de aprendizaje en Devscola, entre 3 y 6 meses dependiendo del alumno, saldrás con las competencias necesarias para trabajar en empresas de software nacionales y de ciudades como Londres.

¿Qué voy a aprender?

  • Construye complejas aplicaciones webcon Ruby on Rails, un potente framework para el desarrollo web.

  • Javascript y jQuery te ayudaran a realizar aplicaciones web dinámicas.

  • Publica tus aplicaciones usando Git de forma rápida sencilla y gratuita.

  • Lo fundamental es siempre importante aprende como hacer que la página web luzca con HTML5.

  • Aprende en MySQL los conceptos de tablas, vistas, IDs, entidades relacionales. etc

  • AGILE AND LEAN

    You will learn modern lean startup principles and agile methodologies used in software development.

¿A quién va dirigido?

Si estas acabando tus estudios y quieres hacer el PFC

O eres un emprendedor que quiere realizar una web app

Para aquellos que han empezado a programar recientemente y quieren formarse para encontrar trabajo
